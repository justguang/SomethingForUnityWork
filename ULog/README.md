##### ULogConfig
	日志配置：
	日志类型（Console、Unity）
	enableLog=true【开启日志】
	enableTime=true【输出日志时显示时间】
	enableTrace=false【输出日志时不显示堆栈信息】
	enableThreadID=true【显示线程ID】
	enableSave=true【保存日志】
	savePath="Ulogs\"【保存路径】
	saveName="ULog.txt"【保存的文件名】
	enableCover=true【覆盖原有日志】
	logPrefix="#"【日志信息前缀字符】
